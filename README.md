## 👍 Features
* Next.js 12.xx
* Css modules

## 👌 Usage

1. Install Dev Depedencies

```sh
npm install
```

2. To start development server

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
